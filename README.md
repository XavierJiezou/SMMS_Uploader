## 文件解释

- `gui.py` GUI模块
- `smms.py` 图床上传模块
- `scanner.py` 文件扫描模块
- `ico.py` ICON文件
- `*save.txt` **上传成功文件列表（自动生成）**
- `*fail.txt` **上传失败文件列表（自动生成）**
- `*links.txt` **CDN链接列表（自动生成）**
- `ib.py` 测试文件（非必要文件）
- `del.py` 自动批量删除模块（非必要文件）
- `ico2b64.py` ICON转换模块（非必要文件）
- `build.bat` EXE文件生成批处理文件（非必要文件）

**自动生成的文件可能随版本更新而变动**
